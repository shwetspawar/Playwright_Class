# Playwright_Class

Collection of Playwright examples, exercises, and test utilities used in the Learning Playwright course.

## Structure
- `Chapter_23_Playwright_Fundamental/` — Playwright project with `package.json` and config
- Other folders contain example scripts and exercises (basic JS, loops, arrays, tasks).

## Setup
1. Install dependencies:

```bash
npm install
```

2. Install Playwright browsers (if not already installed):

```bash
npx playwright install
```

## Running tests

- Run the full test suite:

```bash
npx playwright test
```

- Run a single test file:

```bash
npx playwright test tests/example.spec.ts
```

## Notes
- Test reports are generated under `Chapter_23_Playwright_Fundamental/playwright-report`.
